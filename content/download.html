How to download the RRZE Icon Set

If you want to download the RRZE Icon Set you can clone the whole set from GitHub here. Or you can simply download the complete archive as zip-file.
Feel free to contribute to the set by forking the project via GitHub and sending us pull requests.

For more information visit our Project RRZE-PP/rrze-icon-set on GitHub.

If you don't know yet what GitHub is, visit the help page. It's good 😉
