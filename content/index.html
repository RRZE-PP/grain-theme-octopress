New: The monocrom Icon Set!

The Regional Computing Centre of Erlangen (RRZE) presents a new icon set: monocrom.

MONO, mon.crom, monocrome. Several times the name for the new, monochrome icon set has been changed.
monocrom finally is a new set, which (despite a few exceptions) consists of completely white shapes.
Like the tango styled RRZE Icon Set, monocrom is also specialized for IT topics.
The reason for a new icon set was the revision of several RRZE's websites.

To prevent the the new, clear design from being destroyed by colorful, flashy icons, new icons were necessary.
monocrom was born.
With concrete shapes, abstract processes and functions are shown, which are, though completely colorless, easily understandable for the user.
Bit by bit the icons from the RRZE Icon Set are going to be transfered to the monocrom set. To keep an overview the icons are divided into the following subfolders: actions, emblems, logos and status.


The icons are available in .svg and .png format and can be downloaded directly from the TODO monocrom gallery

The icon set is licensed under the Creative-Commons Attribution Share-Alike licence 3.0.
Therefore interested designers can use the collection for their own creations, starting with this release.

 

About the RRZE Icon Set

The Regional Computing Centre of Erlangen (RRZE) has published a new edition of its open source "symbol collection" online for IT-topics.

We have created a new icon set based on the Tango Desktop Project, which consists of an extensive and very uniformly created icon set. It focuses on Identity Management Systems, especially on user administration.

The collection (see Gallery), which included icons for:

  • different types of users,
  • server applications,
  • status and
  • classifications up to now,

has been extended with numerous icons, especially for participant administration, and has been adjusted to the colours of the Tango Icon Theme Guidelines.

The Icons originate from the RRZE Identity Management Project, where they are used amongst others in the user web front end. The collection is constantly expanded and can be downloaded in "svg" and "png" format.

The icon set is licensed under the Creative Commons Attribution Share-Alike licence 3.0.
Therefore interested designers can use the collection for their own creations, starting with this release.

About the Regional Computing Centre of Erlangen (RRZE)

The Regional Computing Centre of Erlangen has been providing a working IT infrastructure at the Friedrich-Alexander-University of Erlangen-Nuremberg for decades. As the connection between research and engineering, the IT provider sees itself as a centre of competence in computer technology at North Bavaria′s academia as a partner in science. Equally important tasks at the RRZE are the administration and coordination of campus wide IT processes.

Besides the use of commercial software, developing and deploying of Open Source software gets an increasing focus of the RRZE′s attention.
