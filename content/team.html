The Team

A successful project requires many people to play many roles. Some members design graphics, write code or documentation, while others are valuable as testers, submitting patches and suggestions.

Over the time many contributors have actively supported this project. Since this project has a long history starting from an internal project, then being hosted on the not anymore existing BerliOS and now with current home here on GitHub. So there are more contributors then visible online.
All contributions to the project are greatly appreciated.
Get involved today!
